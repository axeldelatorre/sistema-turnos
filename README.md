Sistema de Gestión de Turnos (En desarrollo).

Este proyecto es una solución web para gestionar reservas de pequeños comercios.
Se toma como ejemplo una barberia.

Tecnologías:

SupaBase

React

JavaScript

Tailwind CSS v3.0
